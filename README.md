Proyecto simple de Programación Orientada a Objetos en Java.

## Compilar

```bash
javac -d bin src/*.java
```

## Ejecutar

```bash
java -cp bin App
```

## Código

El código tiene dos clases:

- **Producto**: clase con atributos (nombre, precio, stock, etc.) y métodos (mostrarInformacion, aplicarDescuento, registrarVenta, etc.)
- **App**: clase principal con el método main que crea productos y prueba las operaciones

Este proyecto es un ejemplo básico de programación orientada a objetos. La
clase Producto representa un producto con atributos como nombre, categoría,
precio, stock, disponibilidad e impacto ambiental. La clase App crea varios
productos y prueba las operaciones disponibles.
